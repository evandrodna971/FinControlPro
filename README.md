# FinControl Pro

A premium SaaS for personal and business financial control.

## Tech Stack
- **Frontend**: React, TypeScript, Vite, TailwindCSS
- **Backend**: Python, FastAPI, SQLAlchemy
- **Database**: PostgreSQL / SQLite

## Getting Started

### Backend
```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
